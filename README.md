# Information Gathering 

Usage:
```console
$ python3 info_gathering.py --help
usage: python3 info_gathering.py -d DOMAIN [-s IP]

Basic information gathering tool by Dhairya

optional arguments:
  -h, --help            show this help message and exit
  -d DOMAIN, --domain DOMAIN
                        Enter the domain name for footprinting.
  -s SHODAN, --shodan SHODAN
                        Enter the IP for shodan search.
  -o OUTPUT, --output OUTPUT
                        Enter the file to write output to.
```
